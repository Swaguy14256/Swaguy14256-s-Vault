September 23, 2023 - Long, Temporary Shutdown Of The Site

Well, it has been a while since I posted something here. Over 1 year, in fact. However, there is a reason why I am making this post.

Earlier this year, in January, I found a major privacy issue on this site. It was so major, I HAD to shut down the site until I resolved this privacy issue. As for what this privacy issue was, it was that on free ngrok plans, just tunneling to HTTP Port 80 generates a random link. The problem is that random link includes your IPv4 or IPv6 address. IP addresses have a geographic location assigned to them, and in many cases, this location is the actual location of who's IP address it is. Unfortunately, this site was affected by this fact, which is why I shut it down in the first place.

I did resolve this at like the end of January by finally purchasing a permanent domain for this site. However, I still needed to obtain a better ngrok plan to connect the domain to the ngrok client, and... yeah. Buying a domain for your site and leaving it unused for almost 9 months sure does not seem like a very smart idea, huh. Well, whatever the case is, ngrok DID finally allow free users to have one free "permanent domain". Quotations, because you cannot choose the free domain name, which is why it is still not the actual domain name I chose for this site. Still, it resolves the privacy issue, so I will use it until I can upgrade my ngrok plan.

That is all I need to say. Stay tuned for more updates. See you then.
